 Exploratory Data Analysis (EDA)

This repository contains the completion of Task 3 for the AI & ML Internship. The project focuses on performing deep Exploratory Data Analysis (EDA) on two distinct datasets to understand patterns, detect outliers, and identify key features for future machine learning predictions.

üìä Exploratory Data Analysis Steps Applied
Following the internship guidelines, the following steps were performed for both datasets:

Numerical Distribution: Plotted histograms to understand data spread and skewness.

Categorical Analysis: Used count plots to visualize frequency across types and species.

Outlier Detection: Implemented Box plots to identify anomalies in the data.

Feature Correlation: Created Heatmaps to find strong relationships between variables (e.g., Petal Length vs. Petal Width).

Insight Generation: Documented observations for every visualization.

Feature Importance: Identified which variables are most critical for building prediction models.

üí° Key Insights
Netflix Dataset:
Content production surged after 2015, peaking around 2018-2019.

Movies dominate the platform, comprising over 70% of total titles.

Netflix primarily targets mature audiences (TV-MA).

Iris Dataset:
PetalLengthCm and PetalWidthCm are highly correlated and serve as the best predictors for species.

Iris-setosa is uniquely distinguishable from other species due to its significantly smaller petal size.

The dataset is perfectly balanced with 50 samples per species.
